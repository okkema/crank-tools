# Crank Tools

[Crank Tools](https://crank.tools) is open source bike shop software.

## Features

- [x] About Page
- [x] Transmission Analysis
- [x] Customer Database
- [ ] Service Scheduler
- [ ] Inventory Management
- [ ] Register and Receipts
